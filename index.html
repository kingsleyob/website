My Projects

Custom Unix Shell in C

I developed a custom Unix-like shell in C, designed to enhance the functionality of standard Unix shells with custom features. The project focused on adding core functionalities such as job control, aliasing, and command history to create a more efficient and interactive shell environment.

One of the key features of this shell is its ability to manage jobs in both the foreground and background. This allows users to run commands in the background while working on other tasks, improving multitasking capabilities. Additionally, I integrated piping and I/O redirection, enabling users to chain multiple commands together in a seamless workflow.

To further improve the user experience, I implemented command aliasing and a history recall feature using the readline library. These features allow users to quickly recall and reuse previous commands, saving time and reducing repetitive typing. The project not only helped me improve my C programming skills but also deepened my understanding of Unix-like systems and shell scripting.

Multi-Threaded Patron Rental Simulator (MPRS)

I worked on developing the Multi-Threaded Patron Rental Simulator (MPRS), which simulates a real-time library book rental system. The system supports concurrent server-client interactions, allowing multiple patrons to request books at the same time. The rental process is thread-safe, ensuring smooth and efficient handling of simultaneous requests.

The project includes a dynamic visualization of rental queues using X11 graphics, which represents the real-time flow of patrons and book rentals in a drive-thru style scenario. This makes the system both functional and visually engaging, offering a unique experience for users.

To ensure optimal performance and reliability, I managed inter-process communication and optimized synchronization using semaphores. This was crucial for maintaining a stable and efficient system in a multi-client environment, where race conditions and other edge cases needed to be addressed. The project allowed me to explore multi-threading and server-client communication in depth.
